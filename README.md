# Temperature Guessing Game
A web-based game where players guess the current temperatures in randomly selected European capitals.
Live weather data is retrieved from an external API and compared to the user's input. The closer the estimate, the higher the score.

This project requires a MySQL database connection and an external weather API token.
Database credentials and API tokens are not included for security reasons.

## Technologies
- PHP
- MySQL
- HTML & CSS
- AJAX
